# Module 3: Control Flow

## Module Goal

This module teaches how Dart controls the flow of program execution using conditions, branching, loops, break and continue statements, and assertions.

## Lessons in This Module

1. if, else if, and else
2. switch statements
3. loops
4. break, continue, assertions, and assessment

## Outcomes

By the end of this module, you should be able to:

- write decision-making logic using `if` and `else`
- use `switch` when comparing one value against multiple cases
- repeat actions using `for`, `while`, and `do-while`
- stop or skip loop iterations using `break` and `continue`
- understand what assertions do in Dart
