# Lesson 4: break, continue, Assertions, and Assessment

## Learning Objectives

By the end of this lesson, you should be able to:

- use `break` to stop a loop early
- use `continue` to skip the rest of the current iteration
- understand what assertions do in Dart

## Main Explanation

### `break`

`break` stops the loop immediately.

### `continue`

`continue` skips the remaining statements in the current loop iteration and moves to the next one.

### Assertions

Assertions are checks used during development to catch logic mistakes early.

Example:

```dart
int age = 20;
assert(age >= 0);
