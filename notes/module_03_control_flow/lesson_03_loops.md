# Lesson 3: Loops

## Learning Objectives

By the end of this lesson, you should be able to:

- use `for` loops
- use `while` loops
- use `do-while` loops
- understand when each loop is useful

## Main Explanation

Loops repeat code.

### `for` loop

Use a `for` loop when you know how many times something should repeat.

### `while` loop

Use a `while` loop when repetition should continue as long as a condition stays true.

### `do-while` loop

Use a `do-while` loop when the body should run at least once before the condition is checked.

## Why This Matters

Loops are used for:

- counting
- reading collections later
- retrying actions
- menu systems
- repeated calculations

## Common Mistakes

- writing a loop that never ends
- forgetting to update the loop variable
- choosing the wrong loop for the problem
- making loop conditions confusing

## Best Practices

- keep loop logic simple
- test loop boundaries carefully
- use meaningful loop variable names
- print clearly while learning so you can trace execution
