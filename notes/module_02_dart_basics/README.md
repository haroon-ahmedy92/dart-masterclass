# Module 2: Dart Basics

## Module Goal

This module introduces the core building blocks of Dart programs: syntax, variables, data types, type inference, null safety, constants, operators, comments, input, and output.

## Lessons in This Module

1. Syntax, variables, and data types
2. Null safety and constants
3. Operators, comments, and output
4. Basic input and mini assessment

## Outcomes

By the end of this module, you should be able to:

- declare variables in multiple ways
- use strings, integers, doubles, booleans, and dynamic values
- explain `var`, explicit types, `final`, and `const`
- understand nullable and non-nullable variables
- use common arithmetic and comparison operators
- write readable comments
- read simple terminal input using `stdin.readLineSync()`
