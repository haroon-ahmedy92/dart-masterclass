# Lesson 3: Operators, Comments, and Output

## Learning Objectives

By the end of this lesson, you should be able to:

- use arithmetic operators
- use comparison operators
- use basic logical operators
- write single-line comments
- print output clearly

## Main Explanation

Operators let you perform work on values.

### Arithmetic operators

Common arithmetic operators:

- `+` add
- `-` subtract
- `*` multiply
- `/` divide
- `~/` integer division
- `%` remainder

### Comparison operators

- `==` equal to
- `!=` not equal to
- `>` greater than
- `<` less than
- `>=` greater than or equal to
- `<=` less than or equal to

### Logical operators

- `&&` and
- `||` or
- `!` not
