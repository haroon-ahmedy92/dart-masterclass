# Module 1: Introduction to Dart and Setup

## Module Goal

This module introduces Dart, explains why it is used, helps you set up your environment in VS Code, and guides you through running your first Dart programs.

## Lessons in This Module

1. What is Dart
2. Why Dart is used
3. Setup and tools
4. Running your first program

## Outcomes

By the end of this module, you should be able to:

- describe Dart clearly
- verify Dart installation
- use VS Code for Dart development
- run Dart files from terminal
- understand the purpose of the Dart SDK and `dart` CLI
- write and run simple Dart programs

## Recommended Order

Read the lesson files in order, then run every example in `examples/module_01/`, then complete the assessment in `exercises/module_01/`.
