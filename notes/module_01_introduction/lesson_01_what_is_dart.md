# Lesson 1: What Is Dart?

## Learning Objectives

By the end of this lesson, you should be able to:

- define Dart in a simple and accurate way
- identify where Dart is commonly used
- understand why it is worth learning as a language on its own

## Main Explanation

Dart is a modern programming language used to build fast applications. It is strongly typed, readable, and designed to support productive software development.

Dart is not only for mobile apps. It can be used for:

- command-line applications
- server-side applications
- web applications
- cross-platform app development, especially with Flutter

A beginner should think of Dart as a practical language with clean syntax and strong tooling.

## Why This Matters

Before learning syntax, you should know what role the language plays. This helps you connect code to real development work instead of memorizing rules without context.
