# Lesson 4: Running Your First Dart Program

## Learning Objectives

By the end of this lesson, you should be able to:

- write a very small Dart program
- run a Dart file from the terminal
- understand the role of `main()`

## Main Explanation

A standalone Dart program usually starts execution from the `main()` function.

Example:

```dart
void main() {
  print('Hello, Dart!');
}
