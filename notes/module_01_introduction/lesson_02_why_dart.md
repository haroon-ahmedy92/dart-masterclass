# Lesson 2: Why Dart Is Used

## Learning Objectives

By the end of this lesson, you should be able to:

- explain why developers choose Dart
- describe a few advantages of Dart
- connect Dart to real project work

## Main Explanation

Developers use Dart because it balances readability, structure, and modern features. It supports clean application development and comes with strong official tooling.

Some practical reasons people use Dart:

- readable syntax
- strong typing
- null safety
- support for reusable code
- good command-line tooling
- strong ecosystem for app development

Dart is also a good learning language because it helps beginners build structured habits early.

## Real-World Use

Dart is often used for:

- learning programming structure
- building CLI tools
- creating reusable libraries
- preparing for Flutter development

## Common Mistakes

- learning only by watching and not by running code
- assuming syntax knowledge is enough without tool knowledge
- ignoring how files and folders are organized

## Best Practices

- run every example yourself
- keep your examples small and focused
- write short notes in your own words after each lesson
