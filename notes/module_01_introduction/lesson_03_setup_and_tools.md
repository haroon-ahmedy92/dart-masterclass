# Lesson 3: Setup and Tools

## Learning Objectives

By the end of this lesson, you should be able to:

- verify whether Dart is installed
- explain what the Dart SDK is
- understand the role of the `dart` command
- prepare VS Code for Dart development

## Main Explanation

The Dart SDK is the official development kit for Dart. It includes the libraries and command-line tools needed to write and run Dart applications.

The main command-line tool is `dart`.

Important beginner commands include:

- `dart --version`
- `dart run`
- `dart format`
- `dart create`

## VS Code Workflow

For this course, all examples will be:

- written in VS Code
- run in the VS Code terminal
- saved into GitHub repository

That means you should get comfortable with:
- opening the integrated terminal
- creating files in the correct folders
- running Dart files using `dart run`

## Common Mistakes

- not checking whether Dart is installed
- mixing files into the wrong folders
- copying code without running it
- installing many tools before learning the basics

## Best Practices

- keep VS Code and your repo organized
- run `dart --version` before starting
- use one example file per small concept
- save your work and commit frequently
