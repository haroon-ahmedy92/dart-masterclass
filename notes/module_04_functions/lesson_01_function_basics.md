# Lesson 1: Function Basics

## Learning Objectives

By the end of this lesson, you should be able to:

- define a function
- call a function
- explain why functions are useful

## Main Explanation

A function is a reusable block of code that performs a specific task.

Instead of repeating the same code many times, you can place that logic inside a function and call it whenever needed.

Example:

```dart
void greet() {
  print('Hello from Dart!');
}
