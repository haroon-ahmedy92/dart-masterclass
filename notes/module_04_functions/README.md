# Module 04: Functions

## Module Goal

This module teaches how to define, call, and organize functions in Dart. It covers function syntax, parameters, return values, optional and named parameters, arrow functions, scope, anonymous functions, higher-order functions, and closures.

## Lessons in This Module

1. Function basics
2. Parameters and return values
3. Optional, named, default, and arrow functions
4. Scope, anonymous functions, higher-order functions, closures, and assessment

## Outcomes

By the end of this module, you should be able to:

- define reusable functions
- pass values into functions
- return values from functions
- use optional and named parameters
- write concise arrow functions
- explain local and global scope
- use anonymous functions
- understand higher-order functions and closures at a practical beginner level
