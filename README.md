# Dart Mastery Course

A structured Dart learning repository designed for studying step by step using notes, runnable examples, exercises, and projects.

## Repository Structure

- `notes/` contains lesson notes in Markdown
- `examples/` contains runnable Dart example files
- `exercises/` contains practice tasks and assessments
- `projects/` contains mini projects and larger portfolio projects

## Study Method

For each module:

1. Read the notes
2. Run the examples in VS Code terminal
3. Complete the exercises
4. Commit and push progress to GitHub
